Descripción de los comandos git utilizados en cada punto

Punto 2.

git clone https://github.com/cerebritomultitarea/prueba-git.git

clona el proyecto desde el repositorio de mi cuenta (github.com/cerebritomultitarea)

Punto 3.

git commit -m "se incluye el archivo .gitignore"

se crea el archivo .gitignore, el cual contiene una instruccion para omitir las imagenes con extension jpg ubicadas en la carpeta imagenes

Punto 4.

git commit -m "se modifica el archivo introduccion.html"

se modifica el archivo introduccion.html en el repositorio local

git commit -m "se incluye el marco teorico"

se crea e incluye el archivo marco_teorico.html al repositorio local

git commit -m "se incluye el desarrollo"

se crea e incluye el archivo desarrollo.html al repositorio local

git commit -m "se incluyen las referencias"

se crea e incluye el archivo referencias.html al repositorio local

Punto 5.

git branch diseno

se crea una nueva rama llamada "diseño"

git checkout diseno

se cambia de rama master a rama diseno

git commit -m "se estilizan los enlaces del contenido"

se cambiar el color del texto de los enlaces del contenido y se les quita el subrayado

git commit -m "se agrega efecto hover a enlaces del contenido"

se agrega y estiliza el efecto hover de los enlaces del contenido

git checkout master

se cambia de rama diseno a rama master

git merge diseno

se fusiona la rama master con la rama diseno

git commit -m "se incluye el menu pasos"

se agrega el menu pasos y la pagina que describe los comandos git utilizados en cada punto

git commit -m "fin"

se confirma la version final del proyecto

git commit -m "modificacion fin"

se modifica la version final del proyecto

Punto 6.

git pull origin master

se envia el proyecto al repositorio (github/cerebritomultitarea)
